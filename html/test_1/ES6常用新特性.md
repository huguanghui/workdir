## 最常用的ES6特性
    let, const, class, extends, super, arrow functions, template string, destructuring, default, rest arguments

### 1. let, const
    ES5中只有全局作用域和函数作用域,没有块级作用域.
    let新增具有块级作用域.
    const用于声明常量,一旦声明值就不能改变了.

### 2. class, extends, super
    类,继承机制

### 3. arrow functions
    箭头函数,简化函数

### 4. template stirng
    模板字符串
    反引号\标示开始
    ${}引用变量

### 5. destructuring
    解构

### 6. default, rest
    默认值,扩展符
